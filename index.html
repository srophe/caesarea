About the Application

The Srophé Application is an open source TEI publishing framework. Originally developed as a digital publication platform for Syriaca.org [http://syriaca.org/] the Srophé software has been generalize for use with any TEI publications. Additional features include customizable faceted search and browse functions, D3js visualizations for TEI relationships and a bibliography module with a Zotero API integration.
